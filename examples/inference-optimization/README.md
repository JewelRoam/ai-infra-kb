# 推理优化示例

本目录包含模型推理优化的代码示例。

## 示例列表

| 文件 | 说明 | 技术 |
|------|------|------|
| `onnx_export.py` | PyTorch 模型导出 ONNX | ONNX |
| `tensorrt_optimize.py` | TensorRT 模型优化 | TensorRT |
| `torch_compile.py` | torch.compile 使用示例 | PyTorch 2.0 |
| `quantization.py` | 模型量化示例 | INT8/FP16 |
| `batch_inference.py` | 批量推理优化 | 动态批处理 |
