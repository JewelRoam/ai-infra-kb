# 分布式训练配置示例

本目录包含分布式训练框架的配置示例。

## 示例列表

| 文件 | 说明 | 框架 |
|------|------|------|
| `pytorch_ddp.py` | PyTorch DDP 基础示例 | PyTorch |
| `pytorch_fsdp.py` | PyTorch FSDP 大模型训练 | PyTorch |
| `deepspeed_config.json` | DeepSpeed 配置文件 | DeepSpeed |
| `deepspeed_train.py` | DeepSpeed 训练脚本 | DeepSpeed |
| `accelerate_config.yaml` | HuggingFace Accelerate 配置 | Accelerate |
| `megatron_launcher.sh` | Megatron-LM 启动脚本 | Megatron |
